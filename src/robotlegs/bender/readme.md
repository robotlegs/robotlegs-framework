# Robotlegs

## Framework

The Robotlegs framework:

+ Context
+ Configuration manager
+ Extension installer
+ Guards and hooks
+ Lifecycle
+ Logging

## Extensions

Robotlegs provides a number of built-in extensions:

+ Command map
+ Mediator map
+ View manager
+ And many more

## Bundles

These extensions are combined into convenient drop-in bundles:

+ MVCS Bundle
+ And others

## MXML

Apache Flex integration files.
